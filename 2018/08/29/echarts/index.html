地图数据的获取

  • chinajson使用方法为

    echarts.registerMap('china', chinaJson);

echarts结合使用方法

  • 首先去官网下载echarts文件，引入后，进行初始化容器（echarts.init(xxx))，引入chinajson文件,加载成功后对chart进行配置项设置

  • visualMap可以用来对各省市颜色的配置，show：false则会使展示条隐藏。visualMap有两种方式，连续和分段。如果需要对数据进行精确分区进而准确控制颜色时，需要使用分段方式，具体使用参考echarts文档。

地图响应式布局

  • 正常情况下在window.onresize事件触发时调用chart.resize方法即可。但进行实际操作时无效，后查找到解决方案：

    var resizeWorldMapContainer = function () {
    var mapContainer = document.getElementById('main');
    mapContainer.style.width = window.innerWidth + 'px';
    mapContainer.style.height = window.innerHeight + 'px';
};
//设置容器高宽
resizeWorldMapContainer();
window.onresize = function () {
    //重置容器高宽
    resizeWorldMapContainer();
    chart.resize();
};
